# gTile

gTile is not installed on this Linux Mint Cinnamon system right now. I did not
add blind gsettings writes because there is no local gTile schema to validate.

The included `../cinnamon/keybindings.dconf` gives you an AeroSpace-like
baseline with Cinnamon's built-in window manager:

- `Alt+1..9` switch workspaces.
- `Alt+Shift+1..9` move windows to workspaces.
- `Alt+Shift+h/j/k/l` tile windows.
- `Ctrl+Alt+Shift+h/j/k/l` snap windows.

If you install a Cinnamon-compatible gTile extension later, keep the same
muscle memory:

- use `Alt+Shift+h/j/k/l` for directional placement,
- keep `Alt+1..9` for workspace switching,
- avoid taking over `Alt+Tab`, since Cinnamon already uses it for window
  switching.
